## 1. 프로젝트 소개

스프링과 JPA를 활용한 식당 관리 및 주문 현황을 확인할 수 있는 간단한 서비스 구현.

## 2. 개발 환경

* **Java EE IDE** 
  * Intelli J

* **DB** 
  * H2 DB
  
* **개발 언어** 
  * Java (JDK 1.8)  
  
* **프레임워크**
  * Spring Boot gradle
  
* **형상관리**
  * GitHub
  
  
 
## 3. API 기능 요약

#### 1. **게시판**
 - Store
    1) Store 정보 등록
    2) Store 개별 조회
    3) Store 전체 조회
    4) Store 정보 수정
    5) Store 정보 삭제

 - Order
    1) Order 정보 등록
    2) Order 개별 조회
    3) Order 전체 조회
    4) Order 정보 삭제
